# APP_POIDS_FEMME

Application mobile-first de soutien aux habitudes de vie pour femmes en préménopause, périménopause et ménopause.

Objectifs du MVP :

- reconstruction musculaire douce;
- stabilité glycémique;
- planification de repas;
- liste d’épicerie automatique;
- suivi non culpabilisant;
- expérience adaptée aux profils neurodivergents.

## Installation

```bash
npm install
npm run dev
```

## Vérification

```bash
npm run typecheck
npm run build
```

## Note

Cette application est un outil de soutien aux habitudes de vie. Elle ne remplace pas un avis médical, un diagnostic, un traitement ou un suivi professionnel.
